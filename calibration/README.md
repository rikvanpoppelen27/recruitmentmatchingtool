# Golden set

`golden-set.json` labelt kandidaatГ—vacature-paren met jouw eigen oordeel als
recruiter: `"sterk"`, `"twijfel"` of `"zwak"`. Draai daarna `npm run
calibrate` om te zien of de berekende scores dat oordeel bevestigen.

**De huidige inhoud is een illustratief voorbeeld** (op basis van de test-CV's
uit fase 2 en de echte vacatures uit fase 1) вЂ” vervang dit door je eigen,
echte beoordelingen zodra je meer kandidaten en vacatures hebt.

Elke regel verwijst naar een kandidaat via `candidateEmail` en naar een
vacature via `vacancyCompany` + `vacancyTitle` (zoals je die in het
dashboard/de database ziet вЂ” geen interne database-ID's nodig).
